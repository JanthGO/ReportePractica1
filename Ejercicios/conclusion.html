Conclusión

Las reglas de Grid puede ser utilizadas para diversos propósitos, dependiendo el tipo de página web e podrá escoger una Regla adecuada para dicha página. Esta regla nos facilitará el desarrollo de experiencia de usuario, dando como resultado una vista amigable para el usuario, esto nos dará resultados positivos a la hora de interactuar con el cliente objetivo. Cada regla de grid puede ser adatada a cualquier tipo de contexto, dependerá de la imaginación del desarrollador el uso que pueda aplicarle.
